# PDF Tools

Is a collection of tools for editing PDF files.

It is based on WebAssembly, PWA and does not require a server after deployment.

100% secure because the files never leave the device.

[Host and deploy ASP.NET Core Blazor WebAssembly](https://learn.microsoft.com/en-us/aspnet/core/blazor/host-and-deploy/webassembly).

- [x] Split PDF
- [x] Merge PDF
- [x] Move pages
- [x] Delete pages
- [x] Add signature
